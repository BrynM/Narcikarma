Narcikarma Options

These are the primary options for Narcikarma. If you change any options, you can revert to your saved options with the "Reset Options" button before you save. Your changes will not take effect until you click the "Save Options" button and icon changes won't show until the next refresh. Finally, to restore the default Narcikarma options, use the "Default Options" button.

 
 
 
 
 
 

Settings for Status Flags can be found in the Flag Options tab.

Color settings can be found in the Color Options tab.

Some extras can be found in the Tools tab.

If flags are shown that is... see your "Top Row" and "Second Row" settings in the Main Options tab.

 
 
 
 

You can change the color of each of these by clicking on the color swatch. The slider will change that color's opacity (alpha).
